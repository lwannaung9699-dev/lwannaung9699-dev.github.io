How it works

This automation runs on a fixed daily schedule (twice per day). Each run produces one complete short-form video and publishes it to a single, explicitly-authorized TikTok account. There is no manual editing step - the entire pipeline below runs unattended.

1

Pick the next topic

The system keeps a rotating list of space & astronomy series (e.g. "How Stars Die," "Black Hole Mysteries"), each broken into 2-4 parts. Every run advances to the next part, so related videos post as a connected series.

2

Write the script

An AI language model writes an 8-10 scene narrated script for that topic and part number, including a scroll-stopping opening line and a natural closing line.

3

Generate scene images

An AI image model renders one cinematic illustration per scene, matching the script's description.

4

Synthesize narration

Text-to-speech converts each scene's script line into spoken audio.

5

Assemble the video

All scenes are combined into one vertical (1080x1920) video: smooth Ken Burns-style camera movement, word-by-word burned captions, a background music bed, and cross-fade transitions between scenes.

6

Publish to TikTok

The finished video is uploaded to the authorized TikTok account using TikTok's official Content Posting API (video.publish scope), and the system confirms successful publication before finishing the run.

Who this is for

This is a personal project built and operated by a single developer for their own TikTok account. It is not a service offered to other users, and it does not access or post to any TikTok account other than the one that completed the authorization flow shown during setup.
